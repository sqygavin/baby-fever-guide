# 婴幼儿发热应急管理指南

## 📖 简介

本项目是一个交互式的网页版应急指南，旨在为家长和看护人提供关于婴幼儿发热的清晰、科学的管理建议。它整合了权威的体温分级标准、物理降温方法、退热药物的安全使用剂量与选择，以及明确的就医指引，帮助用户在孩子发热时能够快速获取信息并采取适当的应对措施。

## ✨ 主要内容与功能

本指南通过顶部的导航标签页清晰地划分了各个模块：

1.  **引言 (Introduction)**
    *   简述指南的目的和依据。
    *   包含重要的免责声明。

2.  **体温分级 (Temperature Grading)**
    *   提供详细的发热温度分级表，解释不同发热程度。
    *   通过柱状图直观展示发热温度的各个区间。

3.  **物理降温 (Physical Cooling)**
    *   对比介绍多种安全的物理降温方法及其操作要点和注意事项。
    *   特别强调了应避免的错误做法（如酒精擦浴）。

4.  **退热药物 (Antipyretic Drugs)**
    *   根据不同年龄段推荐合适的退热药物。
    *   提供对乙酰氨基酚和布洛芬按体重计算的详细剂量参考表。
    *   内置 **药物剂量计算器**：用户选择药物类型并输入孩子体重后，可自动计算出推荐的单次用药剂量范围。
    *   包含重要的用药注意事项。

5.  **就医指引 (Medical Guidance)**
    *   列出需要立即带孩子就医的具体情况和标准。
    *   使用环形图（流程示意图）帮助理解发热处理决策流程。
    *   针对特定年龄（如3个月以内婴儿）的发热提供特别提醒。

6.  **结论 (Conclusion)**
    *   总结指南的核心要点，便于快速回顾。

## 🚀 如何使用

1.  **访问指南**:
    *   **本地查看**: 将 HTML 文件（例如 `index.html` 或您保存的文件名）下载到您的设备上，然后使用任何现代网页浏览器（如 Chrome, Firefox, Safari, Edge 等）打开该文件。
    *   **在线访问**: 如果您已将此 HTML 文件部署到网络服务器或 GitHub Pages 等平台，则可以通过相应的网址直接访问。

2.  **内容导航**:
    *   点击页面顶部的导航标签（如“体温分级”、“物理降温”等）即可切换到对应的内容板块。
    *   在屏幕较窄的设备（如手机）上，导航标签栏可以左右滑动以查看所有选项。

3.  **互动体验**:
    *   **图表**: “体温分级”和“就医指引”部分包含由 Chart.js 生成的交互式图表。您可以将鼠标（或手指，在触屏设备上）悬停在图表的特定区域上以查看更详细的提示信息。
    *   **药物剂量计算器**: 在“退热药物”部分，找到“药物剂量计算器”。选择药物种类，输入您孩子的准确体重（公斤），然后点击“计算剂量”按钮，下方将显示推荐的药物剂量范围。
